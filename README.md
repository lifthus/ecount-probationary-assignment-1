## 컨벤션
### 1. 브랜치명
사전에 정의된 타입과 이슈 번호를 붙여서 브랜치를 생성한다.<br/>
e.g. feat/#1/abcd imp/#1/abcd fix/#2/abcd

### 2. 커밋 메시지
브랜치 타입과 연관 이슈 번호를 붙여서 메시지 생성.<br/>
e.g. "feat/#1: 리팩토링"
